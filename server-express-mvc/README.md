# Server Express MVC

Legacy server appplication with Express. Please continue with the main `server-express` version.

## Setup

```sh
npm install
npm run seed
```

## Development

```sh
npm run dev
```

## Test

```sh
npm test
npm run test:unit
npm run test:integration
npm run test:api
npm run test:e2e
```

## Usage

```sh
npm start
npm run start:pm2
```
