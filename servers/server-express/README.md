# Server Express

Server appplication with Express that using intuitive MVC API structure.

## Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Test

```sh
npm test
npm run test:unit
npm run test:integration
npm run test:api
npm run test:e2e
```

## Usage

```sh
# Development
npm start

# Production
npm run start:pm2
```

Then see the [SPECIFICATION](../spec) for more details.

**NOTE:** Do not use Yarn for now, there's an issue with PM2.
